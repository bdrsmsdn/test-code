## Run Locally

Clone the project

```bash
  git clone https://github.com/bdrsmsdn/test-code.git
```

Go to the project directory

```bash
  cd test-code
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Tech Stack

**Server:** Node, Express

**DB:** Mongoose
